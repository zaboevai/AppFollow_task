### Installing

Для старта потребуется:

1) склонировать проект с GitHub `https://github.com/zaboevai/AppFollow_task.git`

2) установить зависимости `requirements.txt`

3) создать базу данных
    
   ```
   flask db init    
   flask db migrate    
   flask db upgrade
   ```
